# Arch Repository Installer (ARI)
Arch Repository Installer installs packages from AUR quikcly and easily.

### Coming features
- Package searching
- Package uninstalling
- ARI uninstalling
- Auto updater

## Installation
```
git clone https://github.com/Marakusa/ari.git
cd ari/
sudo chmod a+x install.sh
./install.sh
```
